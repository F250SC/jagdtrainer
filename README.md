# Jagdtrainer Bayern (PWA) – GitHub Pages ready

## Inhalt (muss im Repo-Root liegen)
- index.html
- styles.css
- app.js
- db.js
- sw.js
- manifest.webmanifest
- assets/

## GitHub Pages (0€)
1) Neues Repo erstellen (z.B. `jagdtrainer-bayern`)
2) Diese Dateien in den Repo-Root hochladen (nicht in Unterordner)
3) GitHub → Settings → Pages
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /(root)
4) Öffnen: https://DEINNAME.github.io/DEINREPO/

## iPhone Installation
1) URL in Safari öffnen
2) Teilen → „Zum Home-Bildschirm“
3) Einmal online starten (damit Offline-Cache gebaut wird)
4) Danach offline nutzbar (Fortschritt bleibt lokal im iPhone gespeichert)

## Karten importieren
Im Setup der App: Import CSV/JSON → „Format anzeigen“.
